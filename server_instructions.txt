# Instrucciones para ejecutar el servidor HTTP y visualizar el CV

1. **Activar el entorno virtual:**
   ```bash
   source /Users/alexi/Documents/GitHub/CV_ALEXI/.venv/bin/activate
   ```

2. **Verificar la versión de Python:**
   ```bash
   python3 --version
   ```
   - Asegúrate de que Python 3.13.5 (o superior) esté disponible.

3. **Iniciar el servidor HTTP:**
   ```bash
   python3 -m http.server
   ```

4. **Acceder al CV en el navegador:**
   - Abre tu navegador y ve a: [http://localhost:8000](http://localhost:8000)

5. **Navegar por las secciones del CV:**
   - Experiencia: [http://localhost:8000/experiencia.html](http://localhost:8000/experiencia.html)
   - Habilidades: [http://localhost:8000/habilidades.html](http://localhost:8000/habilidades.html)
   - Educación: [http://localhost:8000/educacion.html](http://localhost:8000/educacion.html)
   - Competencias: [http://localhost:8000/competencias.html](http://localhost:8000/competencias.html)
   - Equipamiento: [http://localhost:8000/equipamiento.html](http://localhost:8000/equipamiento.html)
   - Referencias: [http://localhost:8000/referencias.html](http://localhost:8000/referencias.html)
   - Contacto: [http://localhost:8000/contacto.html](http://localhost:8000/contacto.html)

6. **Detener el servidor:**
   - Presiona `Ctrl+C` en la terminal para detener el servidor.
